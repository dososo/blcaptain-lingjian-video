# NEUE SACHLICHKEIT · 新客观主义(实物档案)

> 严格的新客观主义(Neue Sachlichkeit)摄影,复现 Karl Blossfeldt / Anna Atkins /
> Bernd&Hilla Becher 的方法本身,不是滤镜、不是致敬。信条:作者退场,让对象自己说话。

## 方法(硬约束,逐条)
- **平光、均匀、无戏剧性打光**:没有硬阴影、没有轮廓光、没有氛围光、没有明暗制造的情绪。光是用来看清楚的,不是煽情的。
- **剔除一切主观与创作表达**:摄影师退场,让对象自己说话。不要"高级感"摆拍、不要电影感调色、不要网红味。
- **通篇清晰**:不要浅景深虚化。
- **高调 high-key**:画面 85% 是浅色,整体明亮。
- **色调:无彩色为主**:白 / 灰 / 黑占绝大部分,只允许**牛皮纸的暗黄褐**作为唯一暖色。不要鲜艳颜色。
- **背景**:均匀的浅灰白工作台面或空白。
- **绝对不要任何文字或字母**(在生成画面里);叙事靠文字卡(代旁白)另叠。
- **关于手**:不禁手。需要人的动作时就正面、清楚地拍手(这个风格靠手撑起「人的在场」)。手要自然解剖结构、五指、朴素、不戴饰品、袖口简洁;不要凭空出现没有身子的手。不需要人的镜头(静物陈列 / 微距材料)就不放手。

## 叙事特征(预热片实证)
无旁白 —— 只有手、物件、光;文字卡代配音,片子不讲道理、它演示。底子可借 1842 蓝晒(cyanotype)的
「摆放 → 曝晒 → 显影」物理过程当母题。

## 给生图 / Seedance 的 prompt 约束
平光均匀无阴影、高调明亮(85% 浅色)、无彩色(白灰黑)+ 唯一暖色牛皮纸暗黄褐、浅灰白工作台背景、
通篇清晰不虚化、画面内无任何文字;要手时正面清楚拍自然五指的手,不要凭空手。
