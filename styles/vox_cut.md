# VOX CUT · Vox 编辑剪影

> 灵剪看家质感。真实照片被切成半调剪影,红标记点出重点 —— 会动的新闻杂志感。

## 气质
橙黑半调、活泼工具感、信息层次清楚;科技评论 / 解释 / 测评的编辑剪影风。

## 视觉锁
- **素材**:网络抓图 + 抠图(rembg 去背)
- **质感**:黑白半调网点 + 朱红错位描边
- **动画**:分帧逐层入场 · 12fps 抖帧(不是匀速推拉)
- **色板**:墨黑 `#1B1B19` · 暖米纸 `#F7F1E4` / `#EFE7D5` · 朱砂 `#E0402A`(深 `#C0361F`)
- **字体**:GlowSansSC(标题·发光大字)+ 思源黑 SourceHanSansSC(正文)+ 等宽 mono
- **引擎**:voxify

## 给生图 / Seedance 的 prompt 约束
真实照片底 → 黑白半调网点处理 + 朱红错位描边点重点;编辑栅格构图、留白放大字标题;
动效走 12fps 抖帧的分帧逐层入场,元素依次入场,不用平滑匀速。
